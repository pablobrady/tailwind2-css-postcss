# TailwindCSS v2 Tryout

<br>
## How to install from scratch

### npm init -y 

### npm install postcss-cli

### npx tailwindcss init -p   
Creates the tailwind.config.js and postcss.config.js

<br>
## Run Commands

+ npm run watch	
+ npm run build:dev
+ npm run build:prod
+ npm run server

<br>
## Optional 

### npm install tailwindcss autoprefixer
